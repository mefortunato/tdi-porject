Historical Stock Prices

This is an example application built using the Flask Python microframework. Enter a stock ticker symbol below to see historical closing prices for the past 5 years. The data is queried from the IEX Financial Data API, converted into a Pandas DataFrame, and an interactive plot is created using Plotly.

An alternative version of this web app was also created using the React javascript framework and can be seen here.
